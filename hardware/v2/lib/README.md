# Shared v2 Hardware Library

This folder is reserved for the common footprint and symbol assets used by every v2 module.

## Shared conventions
- 2x20 stacking header, 0.1" pitch, pin 1 at top-left when the primary side faces you.
- 4x M2.5 mounting holes at the same corner locations on every module.
- 40-pin bus contract defined in `../BUS.md`.

## Planned assets
- 40-pin stacking header symbol and footprint.
- M2.5 mounting-hole pattern.
- Reusable bus or connector symbols shared across modules.
